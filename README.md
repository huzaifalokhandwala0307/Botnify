# BotSniff - AI Bot Detector

BotSniff is a modern, premium web application designed to detect whether an uploaded profile image belongs to a real human or an AI-generated bot.

## Features
- **Premium UI/UX:** Dark mode, glassmorphism, and smooth animations.
- **Client-Side Processing:** All inference is done directly in the browser.
- **Scan History:** Persistent scan results stored locally.
- **Dynamic Model Loading:** Easily swap in your custom TensorFlow.js or Teachable Machine model.

## How to Plug in Your AI Model (IMPORTANT)

The AI model logic is strictly separated from the UI components. 
To add your custom trained model:

1. Open `src/services/modelService.js`.
2. Locate the sections marked `// PLACEHOLDER START` and `// PLACEHOLDER END`.
3. Inside `loadModel`, insert your `tmImage.load(modelURL, metadataURL)` logic.
4. Inside `predictImage`, insert your `model.predict(imageElement)` logic.
5. Remove the mock logic at the bottom of `predictImage` that simulates random results.

## Local Development


## Deployment

This app is configured to be deployed seamlessly on **Vercel**, **Netlify**, or **Hugging Face Spaces**.

- **Vercel / Netlify:** Simply connect your GitHub repository. The `vercel.json` and `netlify.toml` files will handle all SPA routing automatically.
- **Hugging Face Spaces:** Create a Static HTML space, upload the contents of the `dist/` folder (generated by running `npm run build`), or configure the Space to build using Node.
